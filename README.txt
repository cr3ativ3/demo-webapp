Environment requirements:
 - JDK 1.8 installed
 - Internet connection (with access)

Instructions to run WebApp:
 - cmd to folder with this README.txt
 - run command: gradlew tomcatRunWar
 - after server starts go to: http://localhost:8090

Instructions to build WAR:
 - cmd to folder with this README.txt
 - run command: gradlew war
 - built war at: <root>/build/libs/
 
Technologies used:
 - Spring-webmvc 4.3.11.RELEASE
 - AngularJS 1.5.5
 - Bootstrap 4
 - JDK 1.8
 - Gradle 3.1
 - Tomcat 7.x
